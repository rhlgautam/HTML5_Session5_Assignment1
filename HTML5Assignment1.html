This Demo will calculate area of circle of radius value provided by user. A seperate JS file with name 'getArea' has been created and referenced in the head section of HTML file.

Only numbers value will be used in calculation. If user try to enter char/text, it will be give a message 'Please enter a correct value.'

Calculated Area value is rounded off to the nearest integer. Result will be displayed on button click by setting innerHTML of paragraph tag element.
